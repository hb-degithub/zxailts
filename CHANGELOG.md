# 更新日志

所有重要的更新都会记录在这里。

## [1.0.0] - 2026-04-16

### 初始版本

#### 功能
- 💬 多人实时聊天
- 🤖 AI 机器人助手（支持自定义机器人）
- 🌐 公共聊天室、AI讨论区、官方客服群、综合讨论
- 🔐 邀请码注册系统
- 👤 个人中心（头像上传）
- 📊 管理后台（用户、房间、消息统计）

#### 技术栈
- 前端：Vue 3 + Element Plus + Pinia
- 后端：Express.js + MySQL
- 部署：Docker + Docker Compose

#### 部署
- 前端端口：10086
- 管理后台端口：10010
- 后端 API 端口：10020
